# OSI
OSI is a 7 layer model for organizing network communications.

## 7 Layers of OSI
- 7. Application
- 6. Presentation
- 5. Session
- 4. Transport
- 3. Network
- 2. Data Link
- 1. Physical

## 3 Most important layers for cloud work
- 7. Application is important, because the cloud engineer works with requests, that the users make.
- 4. Transport is important, because the ports are on this layer and they are important when choosing the delivery protocols.
- 3. Network is important, because IP addresses and routing live here, which is how AWS VPCs and subnets are structured.
